# 📝 Historial de Cambios - Mx Root Terminal

Todas las versiones notables de este proyecto serán documentadas aquí.

## [1.0.0] - 2026-06-30

### 🎉 Lanzamiento Inicial
- **Instalación:** Script `install.sh` que configura todo automáticamente.
- **Terminal:** Prompt personalizado estilo Kali Linux (rojo y negro).
- **Shell:** Zsh con Oh My Zsh y tema `agnoster`.
- **Herramientas incluidas:**
  - `nmap`, `netdiscover` (Escaneo de redes)
  - `hydra`, `john` (Fuerza bruta)
  - `gobuster`, `nikto` (Análisis web)
  - `curl`, `wget`, `netcat` (Utilidades de red)
- **Scripts propios:**
  - `mx-info` - Muestra información del sistema.
  - `mx-update` - Actualiza todas las herramientas.
- **Plataformas:** Linux nativo y Windows (WSL2).
