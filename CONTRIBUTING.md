# 🤝 Cómo Contribuir a Mx Root Terminal

¡Gracias por tu interés en mejorar Mx Root Terminal! Toda ayuda es bienvenida.

## 📋 Reportar Problemas (Issues)
Si encuentras un error o tienes una sugerencia, abre un *Issue* en GitHub. Por favor, incluye:
- Una descripción clara del problema.
- Pasos para reproducirlo.
- Tu sistema operativo y versión (Linux/WSL).
- Captura de pantalla (si es posible).

## 🚀 Enviar Mejoras (Pull Requests)
1. Haz un *Fork* del repositorio.
2. Crea una rama con tu característica: `git checkout -b mi-nueva-herramienta`.
3. Realiza tus cambios y haz *commit*: `git commit -m 'Agrega script para X'`.
4. Sube tu rama: `git push origin mi-nueva-herramienta`.
5. Abre un *Pull Request* describiendo tus cambios.

## 💡 Ideas para Contribuir
- Añadir más scripts útiles (ej. `mx-wifi`, `mx-scan`, `mx-backup`).
- Mejorar el prompt de Zsh (más colores, iconos).
- Crear una versión con más herramientas de pentesting.
- Escribir documentación más detallada.
- Traducir el README a otros idiomas.

## 📝 Estilo de Código
- Usa **bash** para scripts.
- Comenta tu código en español o inglés.
- Asegúrate de que los scripts tengan permisos de ejecución.

¡Gracias por ser parte de Mx! 🇲🇽
